Infinite Scroll Explained

This infinite scroll is meant to just be a block in the website that goes from one end to the other having text scroll across it infitely. The code is meant to be entirely copy and pasted into the area where you want the infinite scrolling block to go. The text shown can be changed in the il tags. This code uses only HTML and CSS.
